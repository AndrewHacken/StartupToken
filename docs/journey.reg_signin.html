User's journeys for better understanding the way user use a platform.

UJ-001 - User Account Registration

As a user
I want to sign up for StartupToken Platform
So that I'll be able to promote my startup idea, opt for other projects or help as a mentor  

STAGES

Aware

Decide

Confirm

Sign In

Storyboard

  • User get an info about platform from the Landing Page
  • Find Sign Up button and clicks on it
  • User decide what option is better to use for entrance: email or one of  social network account 

  • User goes to email application to confirm user account creation



  • User enters platform with his/her valid account
